# 🤖 AI Resume Analyzer

AI Resume Analyzer is an AI-powered web application that analyzes resumes and provides intelligent insights.

## ✨ Features

- 📂 Resume Category Prediction
- 🛠 Skill Extraction
- 📊 Resume Score (0–100)
- 💡 Improvement Suggestions
- 📄 Downloadable PDF Report
- 🌐 Modern SaaS Dashboard UI

## 🚀 Live Demo

https://ironfist-ai-resume-analyzer.streamlit.app/

## 🛠 Built With

- Python
- Streamlit
- Scikit-learn
- PDFPlumber
- ReportLab
- Pandas
- NumPy
